# EduFlow Hub

Netlify-ready membership website with:
- Google sign-in
- Email + password + verification code
- Profile dashboard
- Admin panel
- Branded broadcast emails
- Netlify Blobs storage

## Recommended Netlify project name
`eduflow-hub-studio`

You can rename the Netlify site after the first deploy.

## Quick deploy
1. Upload all files to a fresh GitHub repository.
2. Import the repository into Netlify.
3. Set environment variables from `.env.example`.
4. Add your new Netlify URL to Google OAuth Authorized JavaScript origins.
5. Redeploy.

See `DEPLOY_0DAN_UZ.md` for the Uzbek step-by-step guide.
