# EduFlow Hub — tez sozlash (yangilangan)

## Bu ZIP ichida nimalar tayyor
- Chiroyli landing page
- Google bilan kirish
- Email + parol + tasdiqlash kodi bilan ro'yxatdan o'tish
- Profil sahifasi
- Admin panel va email broadcast
- Logo, banner, favicon, manifest
- Netlify Functions + Netlify Blobs saqlash qismi

## Oldindan kiritib qo'yilgan qiymat
- `GOOGLE_CLIENT_ID`: kiritilgan

## Endi sizda qoladigan asosiy ishlar
1. GitHub'ga yuklash
2. Netlify'ga import qilish
3. Gmail App Password qo'yish
4. Yangi Netlify domenini Google OAuth origin'ga qo'shish

## Minimal env ro'yxati
- `SESSION_SECRET`
- `ADMIN_ACCESS_CODE`
- `ADMIN_ALLOWED_EMAILS`
- `SMTP_USER`
- `SMTP_PASS`
- `EMAIL_FROM_NAME`
- `REPLY_TO_EMAIL`

## Eslatma
`SITE_BASE_URL` bo'sh bo'lsa ham tizim joriy domenni aniqlashga harakat qiladi.
